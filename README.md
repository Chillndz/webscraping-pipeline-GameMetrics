Titre

GameMetrics — Observatoire de la popularité des jeux vidéo
 
Description

Projet académique réalisé dans le cadre du module Web Scraping à l’ENSEA.

Objectif

Construire un pipeline permettant :

le scraping de données de jeux vidéo

le nettoyage des données

le stockage dans une base SQL

l'exposition via une API REST

l’automatisation avec Celery

la conteneurisation avec Docker

Technologies

Python

BeautifulSoup

Flask

SQL

Docker

Celery

Redis
